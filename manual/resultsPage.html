Once you have run a search, there are three types of results pages you could get:

  1. Author Search Results
  2. Multiple-Author Search Results
  3. Article Search Results

Author Search Results

The Author Search Results Page is split up into two parts:

  1. Co-Authors
  2. Publications
The Co-Authors list will include all of the co-authors the given author has collaborated with. With this list, you can either double click on a new authors name to run a new search, or right click on their name for more options. More options:
  1. Run a new search for that authors name instead of the given author.
  2. Add this author to the search and run a Multiple-Author Search of this author along with the given one.
  3. Copy the authors name to your computer's clipboard so that you can paste it anywhere else you would like.

Multiple-Author Search Results

There are two types of Multiple-Author Search Results Page.

  1. Collaboration Search Page
  2. Co-author Chain Search Page
Collaboration Search Page is returned when more than three author names have been searched together, or there exists a collaboration between two authors.

Collaboration Search Page is divided up into three parts:
  1. Author
  2. Co-authors
  3. List of Collaborations
You can click on author or co-author labels to get a new author result page on the chosen entry. The interaction with the collaboration list is very similar to that of publication list.

Co-author Chain Search Page is divided up into three parts:
  1. First Author
  2. Second Author
  3. The "chain" of authors from the first author to the second author
You can click on author or co-author labels to get a new author result page on the chosen entry. The interaction with the chain list is very similar to that of co-author list. You can also click on the visualize buton to get a visual representation of the chain.
