Author Search

» Back to Getting Started

To search the database for one or more authors, use Author Search.

Using Author Search (Single Author)

This feature displays information about one author. The results include a list of co-authors and a list of publications. Both of these lists have Filter Fields. In addition, the co-authors list has a Graph Visualization button ().

  1. Start at the search page.
    Tip: You can always return to the search page by clicking .
  2. Select the Author Search tab.
  3. Enter a valid author name in the first Author entry field.
    Tip: Use Auto Suggestions to ensure you have entered a valid author.
  4. Press enter or click the search button.

Using Author Search (Two Authors)

This feature displays information about two authors. The results may include a list of publications that the two authors have worked on together.

If the two authors have not worked on any publications together, an Author Chain will be displayed, listing the shortest path from one author to the other in the authorship network. Using the Graph Visualization button () in this case will display the chain.

  1. Start at the search page.
  2. Select the Author Search tab.
  3. Enter a valid author name in the first Author entry field.
  4. Click the plus button () to add another author.
  5. Enter a valid author name in the second Author entry field.
    Tip: If you don't want to search for two authors anymore, use the minus button () to revert to a single author search.
  6. Press enter or click the search button.

Using Author Search (Many Authors)

This feature displays information about multiple authors. The results include a list of publications that the authors have worked on together.

  1. Start at the search page.
  2. Select the Author Search tab.
  3. Use the plus () and minus () buttons to select the desired number of authors.
  4. Enter the author names in the Author entry fields.
  5. Press enter or click the search button.
