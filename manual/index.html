Getting Started with Author Network

Author Network is a tool that can help you search and analyze the relationships between academic co-authors. Have you ever:

  • Wondered which authors a certain author has worked with?
  • Wanted to know which publications certain authors have worked on together?
  • Wished you could view the rich academic authorship network as a graph?
This tool can help you answer all of these questions and more. Read on to find out how.

Features:

  • Author Search
  • Article Search
  • Graph Visualization
  • Filter Fields
  • Auto Suggestions
