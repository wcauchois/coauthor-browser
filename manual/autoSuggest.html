Auto Suggestions

» Back to Getting Started

To use the auto suggestions, all you have to do is follow these easy steps:

  1. Start at the search page and on the Author Search tab.
    Tip: You can always return to the search page by clicking .
  2. Begin to enter an author's name into the entry field.
  3. A drop down menu will appear after you have entered the third character.
    • This is an alphabetic ordering of all authors with those characters in their name.
    • Use your mouse or the arrow keys to scroll through the list
    • Once you have found the author you want, select that name and run the search.
